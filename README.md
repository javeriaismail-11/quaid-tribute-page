# Quaid-e-Azam Muhammad Ali Jinnah — A Tribute Page 🇵🇰

A premium, minimalist, and responsive tribute web page dedicated to the founder of Pakistan, Quaid-e-Azam Muhammad Ali Jinnah. This project highlights his life, legal brilliance, major political milestones, and his ultimate legacy in creating a separate homeland.

## 🚀 Live Demo
You can view the live website here: 
👉 **https://javeriaismail-11.github.io/quaid-tribute-page/**

---

## ✨ Features
*   **Minimalist & Premium UI:** Styled with elegant typography using *Playfair Display* for headings and *Inter* for readable body text.
*   **Comprehensive Biography:** Covers his early life as a barrister at Lincoln's Inn to his role as the first Governor-General.
*   **Interactive Timeline:** A structured milestone tracking from 1876 to 1948.
*   **Embedded Documentary:** Includes a curated YouTube historical documentary section for immersive learning.
*   **Fully Responsive:** Designed with modern layout practices to ensure seamless rendering across desktops, tablets, and smartphones.

---

## 🛠️ Tech Stack Used
*   **HTML5:** Semantic components (`<main>`, `<section>`, `<figure>`) for clean SEO-friendly markup.
*   **CSS3:** Custom layouts, Google Fonts integration, and custom media queries for sleek responsiveness.
